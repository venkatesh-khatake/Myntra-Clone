# Myntra-Inspired UI

## 🚀 Project Overview
This project is a frontend UI replica of Myntra, built using **HTML** and **CSS** for educational purposes. It aims to enhance UI development skills by recreating a modern e-commerce website layout.


## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling
